fruitcupcli
a dumb quick cli made in C++

[under the MIT license](LICENSE)

========================================================
why:

made because i liked linux's terminal, but didn't like linux
so i wanted to make something similar & simpler for windows, so here it is.

the best its for right now is quickly finding folders, apps in common directories you use, and opening them as well.
it's mostly meant for my workflow, but feel free to do whatever with it.

this project is also meant to not feature any external depencies as much as possible to keep me from pulling out my hair when dealing with includes
and also meant to be simple and easy to write for and read.

========================================================
how to use:
setup the config to be able to use apps and directories you use a lot.
the program looks for the config in your Documents folder, so make sure to add it there as fruitcup.txt

once thats done, load it up and enter in help to check the commands.

========================================================
how to compile:
use visual studio 2022 or msvc

========================================================
final note:
if running through command prompt and you have no colors, you'll need to enable it with regedit
https://ss64.com/nt/syntax-ansi.html

there is no AI code whatsoever in this codebase.
creating process code was taken from StackOverflow.
